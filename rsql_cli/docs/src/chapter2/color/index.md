## color

### Usage

```text
.color <on|off>
```

### Description

Controls if output is in color. By default, the color command is set to `on`. If the `color` command is set to `off`,
the output will not be in color.

### Examples

Show the current color setting:

```text
.color
```

Enable color output:

```text
.color on
```

Disable color output:

```text
.color off
```

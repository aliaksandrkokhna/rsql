## header

### Usage

```text
.header <on|off>
```

### Description

The header command sets the display of the header in the CLI. By default, the header is displayed. If the `header`
command is set to `off`, the header will not be displayed.

### Examples

Show the current header setting:

```text
.header
```

Enable the header:

```text
.header on
```

Disable the header:

```text
.header off
```

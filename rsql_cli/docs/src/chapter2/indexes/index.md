## indexes

### Usage

```text
.indexes [table]
```

### Description

Display the indexes for a table. If no table is specified, display the indexes for all tables.

### Examples

Display the indexes for all tables:

```text
.indexes
```

Display the indexes for the `users` table:

```text
.indexes users
```

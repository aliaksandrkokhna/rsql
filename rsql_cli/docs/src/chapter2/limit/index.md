## limit

### Usage

```text
.limit [rows]
```

### Description

Set the limit of rows to be displayed. The default limit is 100.
If the limit is set to 0, then all rows are displayed.

### Examples

Display the current limit setting:

```text
.limit
```

Set row limit to unlimited:

```text
.limit 0
```

Set row limit to 10:

```text
.limit 10
```
